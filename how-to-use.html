About How to use
Everyday use

Set the rhythm, then let it tap you on the shoulder.

Pick an interval Move the reminder slider bar left or right. The app snaps to preset intervals from 5 to 60 minutes and saves your choice automatically.
Start the timer Press Start. The ring counts down and the small status dot lights up while the reminder is running.
Pause or reset Pause stops the current countdown and background notification. Reset stops everything and returns the countdown to the interval you picked.
Respond to the reminder When a reminder fires, read the tip, adjust your posture or take the suggested break, then press Got it to restart the timer. If you ignore it, the app restarts itself after 2 minutes.
Use the chime toggle The music-note button turns the sound on or off. Turning it back on plays a quick preview chime.
Switch themes Use the moon or sun button in the top bar to swap between dark mode and light mode. Your theme is saved on this device.
Notifications

Enable notifications for background reminders.

If notifications are not already allowed, the app shows a banner that says notifications are off. Press Enable to open the browser permission prompt.

When permission is granted and the timer is running, the service worker schedules the next reminder so it can appear while the app is in the background. If notifications are blocked or unsupported, the timer still works while the app is open.

Clicking a notification brings the app back into focus and shows the same reminder tip in the app.

The service worker only schedules reminders after you start the timer. Pause and reset both cancel the pending background reminder.
Install

Add it to your home screen.

This app ships as an installable PWA with a manifest, icons, standalone display mode, and an offline cache. There is no custom install button inside the app.

To install it, open the app in a supported browser and use the browser menu. In Chrome, choose Install app or Add to Home screen. Once installed, it opens in its own app-style window instead of a normal browser tab.

The app caches the main screen, these help pages, the tips file, the manifest, and app icons for offline loading after the first visit.
